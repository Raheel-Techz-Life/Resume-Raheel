## Bottom Line Test

**Before releasing any analysis output, ask:**

1. **Can a decision-maker act on this TODAY?**
   - ✅ Yes → Deploy plan included
   - ❌ No → More analysis needed

2. **Does it include budget estimates?**
   - ✅ Yes → Finance team can allocate
   - ❌ No → Add cost breakdown

3. **Are KPIs measurable and time-bound?**
   - ✅ Yes → Track impact quarterly
   - ❌ No → Define specific metrics

4. **Does it prevent future problems (not just explain past)?**
   - ✅ Yes → Forecasts included
   - ❌ No → Add predictive model

5. **If a stakeholder asks "So what?", do you have a clear answer?**
   - ✅ Yes → Impact articulated
   - ❌ No → Refine the insight

---

**If even ONE answer is ❌, the output is NOT ready.**
